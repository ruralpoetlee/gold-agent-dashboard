# 🏆 黄金价格预测与实时监控智能看板

**基于大模型辅助开发（Vibe Coding）的创新金融数据分析工具。**

本项目致力于为用户提供实时、精准的黄金市场洞察，通过结合前沿的数据抓取技术、智能预测算法以及直观的交互式可视化界面，赋能用户快速把握市场动态。

---

## ✨ 核心功能

-   **实时行情抓取**：
    -   **SGE 国内金价**：实时获取上海黄金交易所的国内黄金价格。
    -   **COMEX 国际金价**：实时追踪 COMEX 国际黄金市场行情。
    -   **汇率数据**：同步抓取人民币对美元汇率，为内外盘溢价分析提供基础。
-   **智能容错机制**：
    -   **周末/休市回退**：集成强大的容错逻辑，在周末休市或数据源无实时更新时，自动回退获取最近一个交易日的收盘价，确保数据流的连续性和页面展示的稳定性，避免“黄页”错误。
-   **内外盘溢价分析**：
    -   **自动计算**：实时计算国内金价与国际金价之间的溢价/折价，揭示套利或市场情绪差异。
    -   **预警机制**：当内外盘溢价超过预设阈值时，触发系统通知（例如 macOS 桌面通知），提醒用户关注。
-   **高级专业图表**：
    -   **Plotly 动态面积图**：利用 Plotly 库生成交互式、美观的黄金历史走势面积图。图表支持平滑曲线和渐变填充，用户可进行缩放、平移等操作，深度分析历史价格趋势。
-   **智能趋势预测**：
    -   **RSI & MACD 算法**：结合多种经典技术分析指标，包括移动平均线（MA）、相对强弱指数（RSI）和异同移动平均线（MACD），构建更精准的趋势预测模型。
    -   **交易信号生成**：根据综合分析结果，给出“强烈看多”、“观望”、“强烈看空”等明确的交易建议，辅助用户决策。
    -   **预测预警**：当系统判定为“强烈看多”时，同步触发系统通知。

## 🛠 技术栈

-   **Python**：项目主要开发语言。
-   **Streamlit**：用于快速构建交互式 Web 应用和数据看板。
-   **Pandas**：强大的数据处理与分析库。
-   **Plotly**：用于生成专业级的交互式数据可视化图表。
-   **akshare**：国内金融数据接口。
-   **yfinance**：国际金融数据接口。
-   **scikit-learn**：机器学习库，用于价格预测。
-   **plyer**：用于跨平台桌面通知。

🚀 如何在本地运行
只需两步，即可在您的本地环境中启动黄金价格预测与实时监控智能看板：

安装依赖：

Bash
pip install -r requirements.txt
启动应用：

Bash
streamlit run streamlit_app.py
运行成功后，您的默认浏览器将自动打开应用界面。
